MacBook查看ip的几个方法

LeBlog
2019-03-02
Common Sense

ifconfig

打开终端

会得到一长串的信息，肉眼寻找即可，大致在en0这个位置

option快捷键

如果只是想单纯的获取到ip地址，更推荐使用快捷键方式。

按住option键，鼠标左键点击wifi标志，弹出的窗口不再是之前选择一系列WiFi的简单列表，而是列出了当前连接网络的详细信息，比如，ip地址 ，频段，速率，噪音等等。
